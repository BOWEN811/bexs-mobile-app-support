Accessibility

Last updated: 2026-07-01

Current Accessibility Support

Pocket Bloom 1.0 is designed for short one-thumb play with large controls, visible focus states, clear status text, and reduced-motion support.

Supported in the First Release

  • Reduced Motion: nonessential animation is reduced when the system setting is enabled.
  • Sufficient Contrast: primary text and controls use high-contrast colors.
  • Differentiate Without Color Alone: game state and important controls use text and icons, not color alone.

Not Claimed Yet

VoiceOver, Voice Control, Larger Text, and Dark Interface support are not claimed for the first release until real-device TestFlight QA verifies those experiences end to end.

Not Applicable

Pocket Bloom 1.0 does not include spoken audio, video, or dialogue content that requires captions or audio descriptions.
