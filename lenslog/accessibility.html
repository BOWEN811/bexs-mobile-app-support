Accessibility

Last updated: 2026-07-01

Current Accessibility Support

LensLog 1.0 is designed with large camera and confirmation controls, visible focus states, clear status text, user confirmation before saving, and reduced-motion support.

Supported in the First Release

  • Reduced Motion: nonessential animation is reduced when the system setting is enabled.
  • Sufficient Contrast: primary text, result confirmation, and data controls use high-contrast colors.
  • Differentiate Without Color Alone: camera state, record type, confidence, and actions use text and icons, not color alone.

Not Claimed Yet

VoiceOver, Voice Control, Larger Text, and Dark Interface support are not claimed for the first release until real-device TestFlight QA verifies camera permission, capture, result editing, save, export, delete, and clear flows end to end.

Not Applicable

LensLog 1.0 does not include spoken audio, video, or dialogue content that requires captions or audio descriptions.
