LensLog

Privacy Policy

Last updated: 2026-07-01

Summary

LensLog is a local-first camera utility for user-confirmed calorie and expense records. The first release does not require an account, does not include advertising, does not include in-app purchases, and does not enable cloud sync.

Camera Use

LensLog uses the camera only when the user chooses to capture a meal or receipt. Photos are used on the device to create an estimate that the user can review and adjust before saving.

On-Device Recognition

On supported iOS devices, LensLog may use Apple's on-device text recognition to read visible receipt text before creating an estimate. The first release does not intentionally upload photos for server OCR or remote AI recognition.

Calorie and Expense Records

Calorie values are estimates and are not medical or nutrition treatment advice. Expense values are estimates and are not accounting, tax, banking, investment, or financial advice. Users must confirm values before saving them.

Local Storage

The first release stores records locally on the user's device. Users can export records, delete a single record, or clear all local records.

Data Collection and Tracking

The first release does not intentionally send photos, calorie records, expense records, or identifiers to a remote server. LensLog does not track users across apps or websites owned by other companies.

Future Changes

If future versions add server recognition, cloud sync, account login, analytics, crash reporting, or third-party integrations, this policy and the App Store privacy labels must be updated before release.
