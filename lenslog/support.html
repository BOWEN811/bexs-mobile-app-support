How to Get Help

For support, bug reports, privacy questions, or estimate-quality feedback, open a public support request and include the app name, device model, iOS version, and a short description of the issue.

Open a support request

Current Release

LensLog 1.0 is a first iOS release candidate focused on camera capture, calorie estimates, expense estimates, user confirmation, local records, export, single-record deletion, and clearing local records.

Important Limits

  • Calorie values are estimates and should be checked by the user.
  • Expense values are estimates and should be checked against the actual receipt.
  • The first release is not a medical, accounting, tax, or financial product.
